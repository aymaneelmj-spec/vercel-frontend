# Happy Deal Transit ERP System

Professional ERP system built for Happy Deal Transit company with comprehensive financial management, inventory tracking, and reporting capabilities.

## System Overview

- **Frontend**: React + Vite with modern UI components
- **Backend**: Python Flask REST API 
- **Database**: SQLite with automatic initialization
- **Authentication**: JWT-based secure login system
- **Features**: Dashboard, Transactions, Invoices, Inventory, Users, Reports

## Quick Start

### Prerequisites
- Python 3.8+ 
- Node.js 16+
- npm or yarn

### Backend Setup
```bash
cd backend
python -m venv venv
venv/Scripts/activate  # Windows
# or: source venv/bin/activate  # Linux/Mac

pip install flask flask-cors werkzeug pyjwt
python run.py
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

### Access the System
- **Frontend**: http://localhost:3000
- **Backend**: http://localhost:5000
- **Default Login**: admin@hdtransit.com / admin123

## Database Schema

- **companies**: Company information

- Use PostgreSQL for production database
